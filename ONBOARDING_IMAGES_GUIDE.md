# Onboarding Images Setup Guide

## Required Images

You need to add 4 images to the `taekwondo-app-/src/assets/` folder:

1. **onboarding1.jpg** - Image showing someone training without results (frustrated look)
2. **onboarding2.jpg** - Image showing proper training with instructor/system
3. **onboarding3.jpg** - Image showing progress/achievement (belt promotion)
4. **onboarding4.jpg** - Image showing action/kick (dynamic taekwondo move)

## Image Specifications

- **Format**: JPG or PNG
- **Dimensions**: Minimum 1080x1920 (portrait orientation)
- **Aspect Ratio**: 9:16 (vertical/portrait)
- **Quality**: High resolution for mobile screens

## How to Add Images

1. Find or create 4 taekwondo-themed images matching the descriptions above
2. Rename them to:
   - `onboarding1.jpg`
   - `onboarding2.jpg`
   - `onboarding3.jpg`
   - `onboarding4.jpg`
3. Copy them to: `taekwondo-app-/src/assets/`

## Temporary Solution (If you don't have images yet)

You can use the same Flashcard.jpg for all 4 slides temporarily:

1. Copy `Flashcard.jpg` 4 times
2. Rename the copies to the names above
3. Replace them later with proper images

## After Adding Images

The onboarding screen will automatically work with the new images. No code changes needed!
